## Simple Example

```fsharp
DropdownMenu().spread props {
    DropdownMenuTrigger() {
        "Open"
    }
    DropdownMenuContent() {
        DropdownMenuLabel() {
            "My Account"
        }
        DropdownMenuSeparator()
        DropdownMenuItem() { "Profile" }
        DropdownMenuItem() { "Billing" }
        DropdownMenuItem() { "Team" }
        DropdownMenuItem() { "Subscription" }
    }
}
```
