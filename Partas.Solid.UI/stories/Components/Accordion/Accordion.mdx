## Composition

You typically compose an accordion in the following manner:

```fsharp
Accordion() {
    AccordionItem(value = "item1") {
        AccordionTrigger() { "Item 1" }
        AccordionContent() { "Content for Accordion Item 1" }
    }
}
```
