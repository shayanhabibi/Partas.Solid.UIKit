You typically compose an accordion in the following manner:

```fsharp
Accordion() {
    AccordionItem(value = "item1") {
        AccordionTrigger() { "Item 1" }
        AccordionContent() { "Content for Accordion Item 1" }
    }
}
```

The Items can be dynamically created:

```fsharp
Accordion().spread acc {
    For(each = [| 1;2;3;4;5;6 |]) {
        yield fun idx _ ->
            AccordionItem(value = $"Accordion{idx}") {
                AccordionTrigger() {
                    $"Title {idx}"
                }
                AccordionContent() {
                    $"Content {idx}"
                }
            }
    }
}
```
