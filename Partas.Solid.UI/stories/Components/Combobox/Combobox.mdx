The usage of a combobox is a bit more involved; however, it is a powerful construct
that is quick to get going when you get the hang of its composition.

#### Minimal Example

```fsharp
Combobox<string>(
    itemComponent = ( fun (itemProps: Kobalte.ItemComponentProps<string>) ->
        ComboboxItem(item = itemProps.item) {
            ComboboxItemLabel() {
                itemProps.item.rawValue
            }
            ComboboxItemIndicator()
        }
        )
    ).spread props {
    ComboboxControl() { yield fun _ -> Fragment() {
        ComboboxInput()
        ComboboxTrigger()
    } }
    ComboboxContent()
}
```
